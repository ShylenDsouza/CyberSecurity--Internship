Cyber Universe Task-Based Cybersecurity Internship
Day 01: Linux Basics & Terminal Operations

Objective of this internship 
The goal of this task was to get comfortable with the Linux command line (CLI), navigate the file system, and perform basic file management operations within a Kali Linux environment.

The commands which I practiced
During this session, I executed over 20 commands to manage files, check system status, and explore network configurations. Key commands included:

* Navigation: `pwd`, `cd`, `cd ..`, `cd ~`, `cd -`
* File Management: `mkdir`, `touch`, `cp`, `mv`, `rm`, `rmdir`
* Listing & Viewing: `ls`, `ls -l`, `ls -la`, `cat`, `head`, `dir`
* System & Network: `history`, `clear`, `whoami`, `ip a`, `nmap`, `sudo ufw`
