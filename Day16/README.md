Defensive Thinking

Attack Surface

Public Wi-Fi Connections:

Public Wi-Fi networks are commonly found in places such as cafes, airports, and public spaces. These networks are often unsecured, which means attackers can monitor or intercept network traffic. Because of this, public Wi-Fi can become a potential entry point for attackers to steal sensitive data or manipulate communication between a user and a website.

Defense 1: Virtual Private Network (VPN)

A VPN encrypts internet traffic between the user’s device and the VPN server. This encryption prevents attackers on the same network from reading or intercepting sensitive information such as login credentials, personal data, or browsing activity.

Even if an attacker captures the network traffic, the encrypted data cannot be easily understood.

Defense 2: Using HTTPS Connections

Websites that use HTTPS encrypt communication between the browser and the web server using SSL/TLS.

This protects sensitive data such as passwords, payment details, and personal information from being intercepted during transmission.

Modern browsers also warn users when they attempt to access websites that do not use HTTPS.

Why One Defense Alone Is Not Enough:

Relying on only one defense can be risky. For example, if a user only depends on HTTPS, attackers might still perform other network attacks such as DNS spoofing or rogue access point attacks.

Similarly, if a VPN connection fails or disconnects unexpectedly, traffic could become exposed.

Using multiple defenses such as VPN encryption along with HTTPS connections ensures that even if one protection layer fails, another layer continues protecting the system. This layered security approach is known as Defense in Depth.

Defense I Previously Underestimated

Previously, I underestimated the importance of using a VPN when connecting to public networks. I believed that simply avoiding suspicious websites was enough to stay safe.

However, I learned that attackers can still intercept network traffic on unsecured networks. Using a VPN provides an additional layer of encryption and significantly improves security when using public Wi-Fi.
