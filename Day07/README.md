Week 1 Reflection

1️. Which task helped you learn the most and why?

The task that helped me learn the most was the Web Basics & HTTP Observation task.
By opening the browser developer tools and observing the network requests and responses, I understood how a browser communicates with a website using HTTP methods like GET. It helped me see how web communication actually works behind the scenes, which is important for understanding web security.


2️. One concept that became clearer for you this week

One concept that became clearer for me is how HTTP requests and responses work.
I learned that when we visit a website, the browser sends a request to the server and the server responds with resources like HTML, CSS, JavaScript, and images. Seeing this process in the developer tools made the concept much easier to understand.


3️ One concept you are still confused about

I am still slightly confused about how networking protocols interact with each other in real-world environments, especially how different layers of the network stack work together during communication. I plan to explore this more while learning networking fundamentals.
